----------------------------------- Prise en main de l'application --------------------------------------------

Bienvenue dans le tutoriel d'utilisation GameLib. Pour lancer l'application, trois solutions s'offrent à vous. 

-La première qui est sûrement la plus simple est de se rendre dans le dossier \GameLib_Csharp\trunk\source\GameLib_Projet\GameLibSetup\Debug
et d'éxécuter le ficher GameLibSetup afin de lancer l'installation de notre application sur votre PC.
Vous aurez le choix de l'emplacement de l'application et un raccourci du même nom
vous sera automatiquement créé sur votre bureau. 

-Pour la deuxième solution, vous devez charger la solution de notre projet dans l'application Visual Studio.
Vous pourrez ensuite lancer l'application à partir du bouton de debuggage de Visual.

-Enfin la troisième est de charger la solution dans Visual Studio, de générer le projet GameLib_Projet et de se rendre
dans \GameLib_Csharp\trunk\source\GameLib_Projet\GameLibProjet_Bin\netcoreapp3.1 et de lancer le fichier GameLib_Projet.exe
sur votre ordinateur. 

Dans cette application, vous devrez créer un compte pour pouvoir ajouter un jeu à vos favoris. 
Pour tester l'application en tant qu'administrateur vous devrez vous connecter sous le pseudo "shotlouf" avec le mot de passe "MotDePasseNoe",
(Nous avons oublier de créer un compte à votre nom). Vous pourrez enfin tester l'ajout et la suppression d'un jeu.

Vous pouvez vérifier la persistance du projet en regardant le fichier Game_Lib.XML se trouvant dans le dossier
\GameLib_Csharp\trunk\source\GameLib_Projet\GameLibProjet_Bin\XML si vous avez utilisez la deuxième et troisème solution. 

Dans le cas de la première il faut ouvrir le fichier XML se trouvant au même endroit que votre éxécutable. 

Nous vous souhaitons une bonne découverte de notre application GameLib. 

L'équipe GameLib.
